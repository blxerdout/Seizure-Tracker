## Usage

Simply open `index.html` in a web browser. No server or installation required.

## Data Storage

All data is stored locally in your browser's localStorage, including:
- Seizure event details
- Video recordings (stored as Base64)
- User notes and triggers

## Privacy

This app runs entirely in your browser. No data is sent to external servers.
